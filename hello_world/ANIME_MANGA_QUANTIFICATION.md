# 만화/애니메이션 정량화

만화를 숫자로 읽을 수 있을까? 에서 시작했다.

## 시작: 만화의 정량화

각본에서 direction:dialogue 비율(발견 17)이 나왔을 때, 자연스러운 질문이 하나 떠올랐다.

> 만화는? 컷 하나에 대사가 얼마나 들어있는지 재면, 지금 이 장면이 그림으로 말하는지 대사로 말하는지 보이지 않을까?

**컷당 대사 정보량** — 이게 만화판 direction:dialogue ratio다.

## 첫 시도: 비전 API로 만화 파싱

테스트 대상: **강철의 연금술사 1화** (완전판, 28페이지)

Claude Opus 비전 API에 만화 페이지 이미지를 던지고, 컷 단위로 파싱시켰다.

```
규칙:
- 읽기 순서: 오른쪽→왼쪽, 위→아래 (일본 만화)
- 컷 = 테두리로 구분된 하나의 그림 영역
- 각 컷: dialogues (speech/narration/thought/sfx), speaker, description
```

**결과:**
- 27페이지 파싱 성공 (1페이지 에러 — 프롤로그+목차 합본 페이지)
- **192컷, 428개 대사** 추출
- 체크포인팅으로 중간에 끊겨도 이어서 가능

파싱된 데이터에 3축 채점 + 9이벤트 분류를 돌렸다 (Claude Opus + GPT-5.2 교차검증).

## 한계: 이거 108화 다 하면?

하가렌은 108화다. 한 화당 ~30페이지. 약 3,000페이지.

비전 API로 한 페이지 파싱에 ~2,400토큰. 3,000페이지면 720만 토큰.
**Opus 기준으로 꽤 아프다.**

수동 파싱도 생각했다. `8컷(대사 없음) 9컷 에드워드` 같은 축약 포맷으로 입력하면 훨씬 빠르긴 한데, 3,000페이지를 사람이 직접?

> 아 되긴 되는데 이거 빡셈

## 전환: 애니메이션 자막

그런데 하가렌에는 애니메이션이 있다. 그리고 자막 파일이 있다.

자막 파일(.ass)에는 이미 **모든 대사가 텍스트로** 들어있고, **타임스탬프**까지 붙어있다.

```
Dialogue: 0,0:00:41.03,0:00:44.24,*Default,NTP,,鋼の錬金術師エドワード·エルリック
Dialogue: 0,0:00:45.00,0:00:48.36,*Default,NTP,,ったく　人使いの荒い大佐だぜ
```

타임스탬프로 할 수 있는 것:
- **대사가 나오는 시간** = dialogue
- **대사가 없는 시간** = direction (그림/액션/침묵으로 말하는 시간)
- → **direction:dialogue ratio가 그냥 나온다**

게다가 대사 텍스트도 있으니 **zlib 압축률**도 바로 측정 가능하다.

만화 한 페이지 파싱하느라 API 부르던 걸, 자막 파일 하나로 5분만에 64화 전체를 끝낼 수 있게 됐다.

## 첫 번째 케이스 스터디: 강철의 연금술사 Brotherhood

### 데이터

| | 일본어 | 영문 |
|---|---|---|
| 에피소드 | 64화 | 64화 |
| 총 대사 줄 | 21,636 | 25,381 |
| 총 글자수 | 230,815 | 645,348 |
| 에피소드 평균 | 3,606자 / 338줄 | 10,083자 / 396줄 |

영문 기준 약 **12.9만 단어** — 각본 약 6.5편 분량.

### 측정 1: 대사 정보 밀도 (zlib)

영문 자막의 zlib 압축률을 할리우드 각본 2,433편 베이스라인과 비교.

```
하가렌 영문 zlib 평균: 0.4479
각본 베이스라인:        0.4105 (std=0.0734)

z-score: +0.51
백분위:  상위 7.4%
```

**대사가 나올 때, 그 대사의 정보 밀도가 할리우드 각본 상위 7.4%.**

64화 전체가 z > 0. 단 한 화도 평균 이하가 없다. 범위: +0.25 ~ +0.84.

에피소드별 특이점:
- **EP 60** (약속의 날 클라이맥스): zlib 최고 (0.4724) — 한마디 한마디가 무거움
- **EP 7** (대사 최다, 5,235자): zlib 최저 (0.4294) — 설명이 많지만 그래도 평균 이상
- **EP 51~56** (최종전쟁): zlib 급상승 + 글자수 급감 — 대사는 줄고, 남은 대사의 밀도가 올라감

### 측정 2: Direction:Dialogue 비율

자막 타임스탬프에서 OP/ED를 제거하고, 대사가 나오는 시간 vs 침묵 시간을 측정.

```
하가렌 direction 평균: 50.1%
각본 베이스라인:        40.5% (std=17.3%)

z-score: +0.56
백분위:  상위 18.5%
```

**화면 시간의 절반 이상을 그림으로 이야기한다.**

에피소드별 변동이 큰데, 이게 서사 리듬이다:

```
EP  7: direction 34.5% — 설명 화 (대사로 세계관 구축)
EP 12: direction 61.2% — 이즈미 회상/감정 (그림으로 전달)
EP 22: direction 58.1% — 윈리의 총 (대사 없는 긴장)
EP 52: direction 65.3% — 전투 (액션)
EP 60: direction 69.7% — 클라이맥스 (거의 70%를 그림으로)
EP 62: direction 66.2% — 그리드 최후 (감정)
```

후반부로 갈수록 direction 비율이 올라간다 = **클라이맥스에서 대사를 줄이고 그림에 맡긴다.**

### 종합: 대사는 적게, 나올 때는 쎄게

| 지표 | 값 | z-score | 의미 |
|------|---|---------|------|
| dial_zlib | 0.4479 | **+0.51** | 대사 정보밀도 상위 7.4% |
| direction% | 50.1% | **+0.56** | 그림 서사 비율 상위 18.5% |

이 두 수치가 동시에 양수라는 것:
- 대사가 적다 (direction이 높으니까)
- 근데 나오는 대사는 밀도가 높다 (zlib이 높으니까)

**불필요한 대사를 안 쓰되, 필요한 대사에 정보를 압축한다.**

## LLM 3축 분석: 64화 전문 일본어 대사

일본어 원문 대사 전체(230,815자)를 반으로 나눠(1-32화 / 33-64화) Claude Opus와 GPT-5.2에 보내고, 합산.

### 3축 점수

| 축 | Opus | GPT-5.2 |
|----|------|---------|
| Information Density | **85** | **87** |
| Payoff Rate | **88** | **88** |
| Direction % | 52% | 54% |
| Balance Score | 9/10 | 8/10 |

### 전반→후반 변화

| | 전반 (1-32) | 후반 (33-64) |
|---|---|---|
| Density (Opus) | 82 | 88 |
| Density (GPT) | 86 | 87 |
| **Payoff (Opus)** | **78** | **93** |
| **Payoff (GPT)** | **82** | **92** |

Payoff가 전반 78-82에서 후반 92-93으로 점프한다. 전반에서 깔아놓은 복선이 후반 클라이맥스에서 **폭발적으로 회수**된다.

### LLM 판단 근거 (전문)

#### Opus — 전반부 (EP 1-32)

**Density 82:**
> 전반부는 세계관 구축(등가교환, 인체연성, 국가연금술사 제도), 캐릭터 동기 확립(형제의 죄와 속죄), 적대세력 도입(호문쿨루스, 스카, 브래들리)을 동시에 진행하며 거의 모든 대사가 서사적 기능을 수행한다. 매화 반복되는 등가교환 내레이션과 일부 코미디 릴리프(키 놀림, 우유 거부)가 밀도를 소폭 낮추지만, 이마저도 캐릭터 특성 강화에 기여한다.

**Payoff 78:**
> 전반부에 심어진 복선 대부분이 32화 이내에서 최소 1차 회수를 달성한다. 현자의 돌 재료, 인체연성 실패물의 정체, 브래들리 정체 등 핵심 셋업이 충실히 회수되며, 연단술·약속의 날 등 일부 장기 복선은 후반부로 이월된다.

**Balance 8/10 (direction 48%):**
> 전반부는 전투(스카전, 제5연구소, 그리드전, 라스트전)와 감정 장면(묘비, 인체연성 회상)에서 대사 없는 액션·연출이 빈번하다. 대사와 비주얼의 교대가 긴장감 조절에 효과적이며, 설명 과잉 없이 시각 서사에 맡기는 비율이 적절하다.

**종합:**
> 전반부 32화는 세계관·캐릭터·적대세력을 동시에 세팅하면서도 복선 회수율이 높아 밀도 있는 서사를 구축한다. 등가교환 내레이션 반복과 코미디 비트가 약간의 여백을 주지만, 후반으로 갈수록 대사가 압축되고 플롯 밀도가 급상승하는 구조가 이미 확인된다. 대사와 비주얼의 균형이 우수하여 액션 애니메이션으로서의 리듬감과 드라마적 깊이를 모두 확보한 전반부이다.

#### Opus — 후반부 (EP 33-64)

**Density 88:**
> 후반부 대사는 국토연성진 발각, 약속의 날 작전, 최종전투, 진리의 문 등 거의 모든 줄이 플롯·캐릭터 아크·세계관을 직접 전진시킨다. 반복적 설명이 일부 있으나(인조인간 능력 재설명 등) 전체 비중은 극히 낮다. 특히 ep50 이후 대사량이 줄면서 한 줄당 서사 기여도가 급격히 상승한다.

**Payoff 93:**
> 33~64화 구간에서 심어진 주요 설정의 90% 이상이 명시적으로 회수된다. 특히 다중 에피소드에 걸친 장기 복선(호엔하임·스카 형 연구서·알의 육체)이 최종전투에서 동시에 수렴하는 구조가 탁월하다. 미회수 요소는 극소수(ex: 유스웰 요키의 서사적 무게 대비 활용도 낮음)에 그친다.

**Balance 9/10 (direction 55%):**
> 후반부로 갈수록 전투·액션·감정 표현이 대사 없이 연출되는 비중이 높아지며, 제공된 계량 데이터(direction 비율 50.1%, 후반부 대사량 감소+zlib 상승)와 일치한다. 대사가 줄되 한 줄의 밀도가 올라가는 구조는 클라이맥스 액션 시퀀스에 최적화되어 있으며, 시각-대사 균형이 매우 적절하다.

**종합:**
> 33~64화는 전반부에서 심어둔 복선이 거의 전부 회수되는 고밀도 수렴 구간이다. 대사량은 줄지만 한 줄당 서사·감정 정보량이 극대화되어 있으며, 호엔하임·스카·그리드·머스탱 등 다수 캐릭터 아크가 동시 완결되는 구조적 완성도가 매우 높다. 시각 연출에 위임하는 비율도 장르 특성에 맞게 적절히 높아, 전체적으로 극본 수준이 상위 5% 이내에 해당하는 우수한 후반부다.

#### Opus — 시리즈 종합

**Density 85:**
> 전반부(82)는 세계관·캐릭터·적대세력을 동시 구축하며 코미디 비트로 소폭 여백을 두고, 후반부(88)는 대사량이 줄면서 줄당 서사 기여도가 급상승한다. 64화 전체로 보면 거의 모든 대사가 플롯·테마·캐릭터 아크를 직접 전진시키며, 반복 설명이나 잉여 대사 비율이 극히 낮은 고밀도 극본이다.

**Payoff 88:**
> 전반부 복선의 대부분이 후반부 클라이맥스에서 동시 수렴하며, 장기 복선(호엔하임·스카 연구서·등가교환 테마)과 단기 복선 모두 명시적 회수율이 90%에 근접한다. 전반부 단독 회수율(78)을 후반부(93)가 강력히 보완하여 시리즈 전체 설계의 완성도가 매우 높다.

**Balance 9/10 (direction 52%):**
> 전반부(48%)는 설명·세계관 구축 대사가 비교적 많고, 후반부(55%)는 전투·감정 연출에 대사 없는 시퀀스가 증가하여 전체 평균 약 52%의 비주얼 비중을 형성한다. 대사와 연출의 교대가 전 64화에 걸쳐 장르 리듬에 최적화되어 있으며, 클라이맥스로 갈수록 시각 서사 위임 비율이 자연스럽게 상승하는 구조가 우수하다.

**종합:**
> 강철의 연금술사 Brotherhood 64화는 전반부의 치밀한 세계관·복선 구축이 후반부 클라이맥스에서 거의 전량 수렴하는, 장편 애니메이션 극본의 교과서적 구조를 보여준다. 대사 밀도·복선 회수율·시각-대사 균형 세 축 모두에서 상위권 점수를 기록하며, 특히 등가교환 테마의 최종 전복(진리의 문 포기)이 서사·캐릭터·주제를 동시에 완결짓는 설계가 탁월하다. 64화 분량 시리즈로서 잉여 에피소드가 사실상 없는, 극도로 효율적인 각본 완성도를 달성한 작품이다.

#### GPT-5.2 — 전반부 (EP 1-32)

**Density 86:**
> 전반부는 '등가교환·금기(인체연성)·현자의 돌·국가연금술사/군'의 핵심 규칙을 반복 노출하면서도, 매 에피소드가 새로운 정보(스카, 인조인간, 연구소, 인주)를 추가해 서사를 전진시킨다. 일상 개그·감정 휴지(우유/키, 가족 대화, 숙소 장면)가 섞이지만 대부분이 인물관계·테마(책임/속죄/전진) 강화로 회수된다. 초반부터 세계관/정치 스릴러 축이 강하게 깔려 정보 손실 구간이 상대적으로 짧다.

**Payoff 82:**
> 초반에 뿌린 설정(현자의 돌, 인조인간, 인주, 군의 어둠)이 EP7~29에서 연쇄적으로 의미를 확장하며 대부분 회수된다. 다만 '연단술만 작동/연금술 불발'의 근본 원인은 1~32화 내 완전 회수보다는 중반 이후로 넘긴다. 전반부는 setup→payoff가 '즉시 회수 + 더 큰 음모로 확장' 구조가 많다.

**Balance 8/10 (direction 52%):**

이게 작가의 역량이다. 통제.

FMA의 맥거핀은 "요키" 1개였다. CG는 3~6개. 특히 세계관 메커니즘(기아스 원리, 라그나레크)이 "그냥 그런 거다"로 넘어간 건 구조적 약점이다.

### 핵심 발견: 이야기 vs 각본

> **Opus**: "복선 설계와 회수의 구조적 완성도에서는 하가렌에 필적하되, 시각-대사 균형에서는 뚜렷한 차이를 보이는 '대사로 이야기하는' 걸작"

코드 기아스의 Payoff는 88로 하가렌과 동급이다. 제로 레퀴엠까지의 설계는 훌륭하다. 하지만 Balance가 6/10 (양쪽 모델 일치).

이게 구분이다:

| | 이야기 (Story) | 각본 (Script) |
|---|---|---|
| 뭘 보는가 | 복선·구조·회수 | **어떻게** 전달하는가 |
| FMA | Payoff 88 | Direction 50%, Balance 9 |
| CG | Payoff 88 | Direction 22%, Balance 6 |

**코드 기아스는 스토리텔러로서는 상위권이지만, 스크립터로서는 중하위권.** 보여줄 수 있는 걸 대사로 설명하는 비율이 너무 높다.

하가렌이 "대사는 적게, 나올 때는 쎄게"라면, 코드 기아스는 **"대사로 다 말하는데, 그래도 할 말은 있다"**.

### 해석: Payoff는 좋다. 하지만.

코드 기아스의 Payoff는 88이다. 제로 레퀴엠까지의 설계는 훌륭하다. 이건 부정할 수 없다.

그리고 이 작품에는 옹호할 점이 있다. 만약 코드 기아스가 액션이나 메카 씬으로 가득 채웠다면? **그건 건담 아류작이다.** 기아스는 "말"의 능력이다. 를르슈의 무기는 주먹이 아니라 전략과 대사다. 코드 기아스 1기는 **피카레스크**다. 주인공이 정의가 아니라는 것을, 를르슈가 직접 말하고, 명령하고, 계산하고, 실수하는 과정을 대사로 따라가야 보여줄 수 있다. 를르슈의 대사는 전체 대사의 **25%에 달한다.** 코드 기아스의 매력은 **를르슈의 압도적인 카리스마에 몰입되는 시청자들**이다. 그 카리스마는 대사에서 나온다.

하지만 그 모든 것을 를르슈의 카리스마와 대사로 이끌고 가는 것은 **구조적인 약점일지도 모른다.**

**스자쿠의 캐릭터를 제대로 잡지 못했다.** 세컨드 주인공이 를르슈의 대칭축이어야 하는데, 대사 비중에서도 심리 묘사에서도 를르슈에 눌린다. 를르슈의 카리스마가 작품을 살린 동시에, 그 쏠림이 구조적 불균형을 만들었다.

**를르슈의 캐릭터성을 꼭 말로만 나타냈어야 했을까?** 장르가 대사를 요구한다는 것과, 이 캐릭터를 대사로만 표현했어야 한다는 건 별개다. 그리고 direction 50%가 꼭 메카씬일 필요는 없다. 를르슈가 침묵 속에서 체스말을 옮기는 손, 회의실에서 오가는 시선, 기아스 발동 전 눈빛의 변화, 군중의 반응, 브리타니아 체제의 압력을 그림으로 보여주는 것 — 이것도 전부 direction이다. 메카씬을 안 넣어도 50%를 채울 수 있었다. 건담 아류작이 되지 않으면서도.

코드 기아스의 진짜 아쉬움은 "정치극이라 대사가 많을 수밖에 없다"가 아니라, **"대사 없이도 정치극을 할 수 있는 여지를 충분히 쓰지 않았다"**는 것이다. 를르슈 한 명의 카리스마에 작품 전체가 기대고 있다는 건, 그만큼 다른 축이 약하다는 뜻이기도 하다.

개인적으로는 S1을 R2보다 좋아하는 편인데, 데이터도 이걸 뒷받침한다. S1 zlib(0.4446, z=+0.47)이 R2(0.4338, z=+0.32)보다 높다. R2는 방영 시간대가 심야에서 주말 오후로 바뀌면서 저연령층을 의식한 스타일 변화가 있었고, 고어 묘사도 상당히 줄었다. 이 외부적 제약이 각본 밀도에도 영향을 줬을 가능성이 높다. S1의 피카레스크 날이 R2에서 무뎌진 건 순수한 각본의 문제만은 아니다.

## 50~60% 밴드 — 그리고 그 한계

할리우드 각본 2,433편에서 이미 확인한 것 (발견 17):

**최적 direction 비율은 약 57%. 하지만 그게 절대적인 수치는 아니다.**

영화 데이터에서도 장르별 편차가 있었다. 57%는 중심값이지 유일한 정답이 아니다. 애니메이션 두 케이스 스터디가 이걸 다시 확인해준다:

| 작품 | direction% | 장르 | 밴드 안? |
|------|-----------|------|---------|
| FMA Brotherhood | 50.1% | 액션/모험/다크판타지 | **안에 있음** |
| Code Geass | 23.5% | 피카레스크 정치극 | **밖에 있음** |

FMA는 밴드 안에서 작동한다. 보여줄 것은 보여주고, 말할 것은 말하는 균형.

코드 기아스는 밴드 밖이다. 하지만 앞서 해석했듯이, 이 작품에 direction 50%를 요구하는 건 건담 아류작을 만들라는 소리다. 를르슈의 카리스마가 대사에서 나오는 작품에서 대사를 줄이면 작품이 죽는다.

그러면 밴드의 의미가 정리된다:

> **50-60% 밴드는 범용 최적이 맞다. 다만 그 50%를 무엇으로 채우는가는 장르마다 다르다.**
> 액션물은 전투와 액션으로, 정치극은 침묵·표정·시선·상징으로 채울 수 있다.
> 코드 기아스가 밴드를 벗어난 건 "정치극이라서"가 아니라, 대사 없이 정치극을 하는 방법을 충분히 활용하지 않았기 때문이다.

코드 기아스의 Balance 6/10은 장르의 숙명이 아니라, 연출 선택의 결과다.

### 해석: 서사 구조의 프랙탈 분석

프리렌의 전체 서사 구조는 평균 이하다. 정확히는 **고전 왕도물의 특성을 정확히 따른다.** 목표(오레올)를 향해 여행하고, 동료를 모으고, 시험을 치르고, 적과 싸운다. 나쁜 각본이라고 볼 수는 없지만, 좋은 각본은 절대 아니다. zlib -0.23, Density 78이 그걸 말한다.

근데 우리는 프리렌을 본다. 왜?

이야기가 전개된다는 건 **셋업→페이오프**다. 복선이 쌓이고 회수된다. 하가렌이 전반부 payoff 78 → 후반부 93으로 대수렴하는 게 전형적인 좋은 각본이다. 그러면 페이오프가 아직 안 온 셋업 구간을 왜 볼까?

답을 찾기 위해 3작품의 자막 텍스트에서 **엔트로피 시계열**을 추출하고, **DFA(Detrended Fluctuation Analysis)**로 프랙탈 지수를 측정했다.

#### 방법

전체 대사를 시간순으로 연결 → 슬라이딩 윈도우(500자, stride 100)로 zlib 엔트로피 시계열 생성 → DFA로 Hurst exponent(α) 산출.

- α ≈ 0.5: 무상관 (백색 잡음 — 스케일 간 관계 없음)
- α ≈ 1.0: **1/f 잡음 (완벽한 프랙탈 — 어느 스케일에서 잘라도 같은 패턴)**
- α ≈ 1.5: 브라운 운동 (일방향 추세)

#### 결과

| 작품 | 화수 | DFA α | 멀티스케일 기울기 | 해석 |
|------|------|-------|----------------|------|
| **Frieren** | 28 (연재중) | **0.957** | -0.384 | **거의 완벽한 1/f 프랙탈** |
| FMA Brotherhood | 64 (완결) | 0.725 | -0.416 | 약한 프랙탈 |
| Code Geass | 50 (완결) | 1.142 | -0.118 | 브라운 운동 (일방향 가속) |

#### 세 가지 서사 전략

**1. 코드기아스 — 일방향 가속형 (α=1.14)**

긴장이 한 방향으로 계속 쌓인다. 에피소드마다 위기가 에스컬레이션되고 시즌 끝에서 터진다. 셋업과 페이오프가 **시간적으로 분리**되어 있다. 전반부는 셋업, 후반부는 폭발. 매 화의 클리프행어가 다음 화로 긴장을 넘기는 구조. 멀티스케일 σ가 거의 안 줄어듦(기울기 -0.12) — 대규모 추세가 모든 걸 지배.

**2. FMA Brotherhood — 레이어드 수렴형 (α=0.73)**

에피소드마다 미니 페이오프가 터진다(터커 사건, 휴즈 사망, 그리드전 등). 그러면서 대형 복선이 겹겹이 쌓여서 후반부에 동시 수렴한다(payoff 93). 로컬 스케일과 글로벌 스케일에서 **서로 다른 패턴**이 작동한다 — 에피소드 단위는 자기완결적이고, 시리즈 단위는 누적형. 그래서 α가 0.73으로 "약한 프랙탈" — 프랙탈이긴 하나 완전하지 않다. 멀티스케일 σ가 빠르게 줄어듦(기울기 -0.42) — 로컬 변동이 크고 글로벌은 안정적.

**3. 프리렌 — 프랙탈형 (α=0.96)**

α=0.957. 거의 정확히 1.0. **어느 스케일에서 잘라도 같은 구조.**

할머니가 꽃밭 이야기를 해주는 씬 하나가 이미 완결된 감정 사이클이다. 동시에 그 씬은 "1000년을 산 엘프가 인간을 이해해가는" 에피소드의 축소판이다. 그 에피소드는 "오레올을 향한 여정" 전체의 축소판이다. **씬 = 에피소드 = 시리즈.** 같은 패턴이 모든 스케일에서 반복된다.

#### "셋업 구간을 왜 보냐?"에 대한 답

**프리렌에는 셋업 구간이 없다.**

코드기아스에는 셋업 구간이 있다 — 긴장이 쌓이는 전반부와 터지는 후반부가 분리되어 있으니까. FMA도 있다 — 에피소드 단위로는 자기완결적이지만, 시리즈 단위로는 셋업이 누적되니까. 프리렌은 매 순간이 그 스케일에서의 완결된 감정 사이클이면서 동시에 더 큰 스케일의 일부다. **프랙탈은 부분이 전체를 닮는 구조**이기 때문에, 1화만 봐도, 3화만 봐도, 28화를 다 봐도 **같은 경험**을 한다.

이게 프리렌이 zlib -0.23, Density 78이면서도 작동하는 이유다. 정보량이 적은 게 아니라, **모든 스케일에서 균등하게 분배**되어 있는 것이다.

#### 좋은 전개란 무엇인가

세 작품 모두 좋은 작품이다. 하지만 "좋은 전개"의 방식이 완전히 다르다.

| | CG (α=1.14) | FMA (α=0.73) | Frieren (α=0.96) |
|---|---|---|---|
| 전략 | 일방향 가속 | 레이어드 수렴 | 프랙탈 |
| 셋업↔페이오프 | 시간 분리 | 스케일 분리 | **분리 없음** |
| 왜 재밌나 | 다음 화가 궁금 | 매화 터지면서 쌓임 | 매 순간이 완결+미완 |
| 약점 | 초반 진입장벽 | 후반 의존도 | 큰 폭발이 없을 수 있음 |
| 리스크 | 후반 수습 실패 시 붕괴 | 수렴 실패 시 허무 | 프랙탈이 깨지면 "늘어짐" |

**28화 전부 has_complete_cycle = true.** 단 하나의 에피소드도 순수 셋업 회차가 없다.

| 아크 | 화수 | Payoff | 핵심 |
|------|------|--------|------|
| 프롤로그 (힘멜→하이터) | 1-2 | 74 | 후회 → 새 목적 (인간 이해) |

GPT-5.2의 결론:

> "Instead of 'pure setup → later payoff,' it uses **'setup+payoff now → plus carry-forward,'** producing a steadier cadence of closure and meaning rather than a long ramp to a single delayed release."

**"지금 셋업+페이오프 → 그리고 이월."** 이것이 프랙탈 서사의 작동 원리다. 씬 하나가 완결된 감정 사이클이면서 동시에 더 큰 아크의 일부이고, 그 아크도 시리즈 전체의 축소판이다. 부분이 전체를 닮는 구조.

이 분석은 앞의 DFA α=0.957(거의 완벽한 1/f 프랙탈)과 정확히 일치한다. 엔트로피 시계열이 프랙탈 구조를 보이는 이유가, 실제로 매 에피소드마다 셋업과 페이오프가 동시에 일어나기 때문이었다.

#### 좋은 전개는 전부 프랙탈이다

프리렌만 프랙탈인 줄 알았다. 틀렸다.

|---|---|---|---|
| payoff 평균 | **72.3** | 69.0 | 66.1 |

σ가 셋 다 거의 같다. 코드기아스는 프리렌과 마찬가지로 전화 사이클 100%. FMA도 63화 중 단 1화(ep60, "약속의 날" 직전 순수 에스컬레이션)만 빠졌다.

**셋 다 프랙탈이다.**

차이는 프랙탈이냐 아니냐가 아니라 **프랙탈의 진폭**이 다르다:

|---|---|---|---|

| Code Geass | 40 (R2E10 중간정리) | 95 (R2E25 제로 레퀴엠) | 가장 넓은 진폭. 그런데 매화 반드시 닫는다 |

처음에 이 세 작품의 전개를 이렇게 분류했다:

- CG: 일방향 가속 (α=1.14)
- FMA: 레이어드 수렴 (α=0.73)
- Frieren: 프랙탈 (α=0.96)

이건 **맛의 차이**를 구조의 차이로 착각한 것이었다. DFA α가 다른 건 맞다. 하지만 에피소드별 payoff를 뜯어보면 **셋 다 매화 setup→payoff 사이클을 닫고 있다.** 프랙탈의 정의 — "부분이 전체를 닮는 구조" — 를 셋 다 만족한다.

진짜 차이는 프랙탈이냐 아니냐가 아니라, **매화 해소량 vs 이월량의 비율**이다. 정도의 차이일 뿐이다.

| | 매화 해소 | 이월 | 결과 |
|---|---|---|---|
| Frieren | 多 | 少 | 어느 화를 봐도 만족. 바닥 52 |
| FMA:B | 中 | 中 | 초반 독립 미니아크 강함 → 후반 연쇄 폭발 |
| Code Geass | 少 | 多 | 마지막에 한번에 해소. 바닥 40, 천장 95 |

코드기아스의 payoff 최고화가 R2E25(95), R2E21(85), R2E13(88)으로 전부 후반에 몰려 있고, 최저화가 S1E03(45), S1E10(45), R2E10(40)으로 전반~중반에 몰려 있는 것이 이걸 그대로 보여준다. 이월량이 많고, 뒤에서 한 번에 터뜨린다. 프리렌은 반대로 어디를 잘라도 비슷하게 해소된다.

#### 영화와 드라마의 차이, 그리고 서스펜스

Part 3에서 서스펜스를 이렇게 정의했다:

- **서스펜스** = 심은 걸 회수하는 것.
- **맥거핀** = 극을 이끌지만 회수되지 않는 것.

그리고 **"서스펜스에서 가장 중요한 건 설치가 아니라 회수다."** Payoff Rate가 서스펜스의 본질이고, 설치 방법은 스타일이다.

이 정의를 지금 데이터에 대입하면: GPT-5.2로 측정한 payoff score가 곧 서스펜스다. 매화 사이클을 닫는다 = 매화 서스펜스를 성립시킨다. 프리렌, FMA, 코드기아스 셋 다 이걸 하고 있다.

**그런데 영화와 드라마의 구조적 차이가 있다.** 영화는 2시간 안에 전부 심고 전부 회수해야 한다. 선택지가 없다. 드라마는 **이월할 수 있다.** 다음 화에 넘길 수 있다.

이 "이월할 수 있다"가 핵심이다. 영화에는 없는, 드라마만의 자유도.

그리고 이 자유도가 양날의 검이다. 복선을 회수하지 않으면 서스펜스가 성립하지 않는다. 서스펜스가 성립하지 않으면 "전개가 구리다." 구린 드라마는 드라마라는 포맷이 허락하는 이월을 **남용**한다. "다음 화에서 해결될 거야"만 반복하면서 안 닫는다.

**그래서 닫아야 한다.** 1화 자체로도, 아크 자체로도, 완결성을 가져야 한다. 좋은 드라마와 구린 드라마의 차이는 이걸 하느냐 안 하느냐일 뿐이다. 프리렌이 매화 닫고, FMA가 매화 닫고, 코드기아스가 매화 닫는 것은 — 좋은 영화가 2시간 안에 닫는 것과 같은 원리를 에피소드 단위에서 실행하는 것이다. 정도의 차이일 뿐.

| | 영화 | 좋은 드라마 | 구린 드라마 |
|---|---|---|---|
| 이월 가능 여부 | 불가 | 가능 | 가능 |
| 매 단위 payoff | 필수 (2시간 내) | **한다** (매화/매 아크) | **안 한다** |
| 서스펜스 성립 | 자동 | 자동 | **불성립** |
| 관객 반응 | — | "전개 좋다" | "전개 구리다" |

#### 실험: Steins;Gate — "끝까지 보면 재밌는데 초반 전개 구리다"

가설을 검증한다. 구조가 안 좋은 드라마는 매화 서스펜스를 닫지 않을까?

슈타게는 완벽한 실험 대상이다. "좋은 작품이지만 초반 전개 구리기로 유명한" 케이스. 이미 z_dir=-0.29로 좋은 그룹 최하위였다.

GPT-5.2 payoff 분석 결과 (영문 자막 23화, 14화 누락):

| | 1-12화 (전반) | 13-24화 (후반) | 차이 |
|---|---|---|---|
| payoff 평균 | **58.3** | **72.7** | **+14.4** |
| 범위 | 35~75 | 55~92 | — |
| 사이클 완결 | 12/12 | 11/11 | 23/23 (100%) |

4작품 비교:

| | 전체 avg | 최저 | 최고 | σ |
|---|---|---|---|---|
| Frieren | **72.3** | **52** | 92 | 10.9 |
| FMA:B | 69.0 | 45 | **95** | 10.6 |
| Code Geass | 66.1 | 40 | **95** | 11.8 |
| Steins;Gate | 64.8 | **35** | 92 | **13.7** |

슈타게의 σ가 13.7로 가장 크다. 전반/후반의 낙차가 가장 극적이라는 뜻이다.

ep1 = 35. 지금까지 측정한 "좋은 작품" 중 최저 바닥.

GPT-5.2는 이 에피소드에 has_complete_cycle=true를 줬다. 근거: "오카베가 문자를 보냄 → 그 문자의 발신일이 5일 전인 걸 발견." 하나라도 회수됐으니 닫았다는 판단이다.

**이건 닫은 게 아니다.**

ep1의 carries_forward를 보면: 쿠리스 살인, 위성 추락, 젤바나나, Reading Steiner, 존 티토, 메탈 우파, SERN — 7개. 실제로 닫힌 건 "문자 타임스탬프 발견" 하나. 그리고 그 "발견"조차 관객에게는 "그래서 뭐?"다. 이걸 서스펜스의 해소라고 할 수 없다. (이거 때문에 슈타게 초반부를 다시 다 봤다.)

프리렌의 최저는 52(ep18, 시험 등록). 이 에피소드에서도 "팀 배정 → 첫 전략 수립 → 시험 규칙 파악"이라는 명확한 미니 사이클이 있다. FMA의 최저는 45(ep48)인데, 쿠데타 준비 화임에도 각 진영의 위치가 확정되는 구조적 해소가 있다. 아무리 이월한다 해도, 이 둘은 50 근처에서 무언가를 닫는다.

**35는 닫은 게 아니다. 붕 뜬 것이다.**

물론 애니메이션은 장기 프로젝트고, 매화 긴장과 해소를 균등하게 배분할 수는 없다. 이월 자체가 나쁜 게 아니다. 하지만 아예 붕 떠버리는 화가 생기면 — payoff가 40 이하로 내려가면 — 관객이 "전개가 안 풀린다"고 느낀다. 그게 슈타게 초반부에서 일어난 일이다.

payoff 기준으로 다시 세면:

| payoff 구간 | 슈타게 전반 (1-12) | 해석 |
|---|---|---|
| **≥ 60 (닫힘)** | ep5(65), ep6(70), ep7(70), ep10(65), ep11(75) = **5화** | 관객이 만족하는 화 |
| **40-59 (형식적)** | ep2(45), ep3(55), ep4(60), ep8(60), ep9(55), ep12(45) = **6화** | 뭔가 움직이긴 하는데 "그래서?" |
| **< 40 (붕 뜸)** | ep1(35) = **1화** | 안 닫힘 |

12화 중 실질적으로 닫힌 건 5화. 나머지 7화는 관객이 "언제 재밌어지냐"고 묻는 구간이다.

후반(13-24화)은 전부 55 이상, 평균 72.7. 프리렌 전체 평균(72.3)과 거의 동일하다. **슈타게 후반부는 프리렌급이다.** 문제는 거기까지 가는 12화 동안 관객이 이탈한다는 것.

이건 "구린 작품"과는 다른 패턴이다. 슈타게는 후반에 회수한다. 진짜 구린 작품은 끝까지 안 회수한다. 하지만 초반 12화의 payoff 부족은 사실이고, 그래서 "끝까지 봐야 재밌다"는 평이 따라다닌다.

#### 실험: Platinum End — 진짜 구린 전개

같은 GPT-5.2로 플래티넘 엔드 24화를 측정했다.

| | Frieren | FMA:B | Code Geass | Steins;Gate | **Platinum End** |
|---|---|---|---|---|---|
| avg | **72.3** | 69.0 | 66.1 | 64.8 | **61.9** |
| σ | 10.9 | 10.6 | 11.8 | 13.7 | **14.8** |
| min | **52** | 45 | 40 | 35 | **35** |
| payoff < 50 비율 | **0%** | 5% | 12% | 22% | **29%** |

GPT-5.2의 절대값은 관대하다. "상황 변화"를 payoff로 치고, "동맹 결성"이나 "규칙 시연"을 닫힌 것으로 인정한다. 하지만 **같은 기준으로 전부 쟀으니 상대 비교는 유효**하다.

#### 이월 주기: 진짜 차이가 드러나는 곳

payoff 절대값보다 중요한 게 있다. **빚을 진 뒤 얼마나 빨리 갚느냐.**

코드기아스 R2E10-13:

> R2E10(**40**) → R2E11(70) → R2E12(72) → R2E13(**88**)

40에서 넘어갔지만 바로 다음 화 70, 그리고 3화 뒤 88로 폭발. 이월한 빚을 이자까지 붙여서 갚는다. (R2E10을 직접 봤다. 안 닫힌다. 근데 R2E11에서 확실하게 닫는다.)

슈타게 전반:

> ep1(**35**) → ep2(45) → ep3(55) → ep4(60) → ep5(65) → ep6(70) → ep7(70)

서서히 올라가긴 한다. 근데 ep1에서 이월된 것 — 쿠리스 살인, 위성 추락, SERN, Reading Steiner — 이 중 **하나도 12화까지 안 터진다.** 12화 동안 빚만 쌓이고, 13화 마유리 사망에서 비로소 전부 폭발.

플래티넘 엔드:

> ep1(70) → ep2(55) → ep3(60) → ep4(45) → ep5(50) → ep6(**35**) → ep7(55) → ep8(80) → ep9(60)

들쭉날쭉. 방향이 없다. 올라가다 떨어지고, 또 올라가다 떨어지고. 빚을 갚다가 또 지고 갚다가 또 진다.

#### 좋은 전개의 세 가지 전략

| 전략 | 작품 | 패턴 | 관객 체감 |
|---|---|---|---|
| **매화 닫기** | Frieren | 바닥 52. 빚이 없다 | "잔잔한데 왜 계속 보게 되지?" |
| **이월 후 즉시 폭파** | Code Geass, FMA | 셋업 화 있음. 근데 다음 화에 이자까지 갚음 | "못 끊겠다" |
| **장기 축적 후 일괄 폭파** | Steins;Gate | 12화 빚 → 13화 폭발 | "12화까지 참아" |

세 전략 모두 **결국은 닫는다.** 정도의 차이일 뿐이다.

구린 전개는 전략이 아니다:

| | Platinum End |
|---|---|
| 패턴 | 방향 없음. 빚 → 갚음 → 빚 → 갚음 반복 |
| 관객 체감 | "전개가 구리다" |

프리렌이 잔잔해 보여도 우리가 계속 보는 이유 — 잔잔한 만큼 **매번 닫기 때문이다.** 코드기아스와 하가렌은 중간중간 셋업도 한다. 근데 적어도 다음 화에 확실하게 닫는다. 아예 터뜨려서. 슈타게는 12화 동안 안 닫는 대신 후반에 프리렌급(avg 72.7)으로 전부 회수한다.

**"좋은 전개"는 닫는 것이고, "나쁜 전개"는 닫지 않는 것이다.** 닫는 주기와 강도가 작품의 맛을 결정할 뿐.

#### Payoff 분포의 Shannon Entropy

이걸 엔트로피로 확인한다. 에피소드별 payoff score를 10점 구간(30s, 40s, ..., 90s)으로 나누고 Shannon entropy를 계산하면:

| 작품 | H_norm | 분포 (30s~90s) | 해석 |
|---|---|---|---|
| Frieren | **0.712** | 0, 0, 4, 4, **12**, 7, 1 | 70대에 집중. 예측 가능하게 안정적 |
| FMA:B | 0.779 | 0, 2, 8, 17, **24**, 9, 3 | 60-70대에 집중 |
| Code Geass | 0.802 | 0, 5, 9, 10, **19**, 6, 1 | 약간 퍼짐 |
| Steins;Gate | 0.864 | 1, 2, 3, 6, **8**, 2, 1 | 전반/후반 낙차 |
| **Platinum End** | **0.940** | 2, 2, 6, 5, 5, 2, 2 | **거의 균등분포** |

(H_norm: 0=한 구간에 완전 집중, 1=모든 구간에 균등 분포)

플래티넘 엔드의 H_norm = 0.940. 거의 1.0. payoff가 30대부터 90대까지 고르게 흩어져 있다. **방향이 없다는 걸 엔트로피가 그대로 잡는다.**

프리렌은 0.712. 28화 중 12화가 70대에 몰려 있고 30-40대는 0화. 매화 비슷한 수준으로 닫으니까 분포가 집중된다.

**낮은 payoff 엔트로피 = 좋은 전개.** 매화 안정적으로 닫으니까 분포가 좁고, 구린 전개는 아무 데나 흩어지니까 엔트로피가 높다.

#### TV 애니메이션/드라마의 원칙

결국 영화와 같은 원리다. 스케일만 다르다.

영화는 2시간 안에 닫아야 한다. TV는 **1화 안에** 닫아야 한다. 이월할 수 있다는 건 "안 닫아도 된다"가 아니라 "닫으면서 동시에 다음 것도 깔 수 있다"는 뜻이다.

GPT-5.2가 프리렌에 대해 한 말이 정확히 이거다:

> **"setup+payoff now → plus carry-forward"**

지금 닫고 → 그러면서 이월도 한다. **닫는 게 먼저고, 이월은 부산물이다.** 순서가 반대면 — 이월을 먼저 하고 닫는 건 나중에 — 그게 슈타게 초반이고, 아예 방향 없이 하면 플래티넘 엔드다.

#### 실험: 빙과(氷菓) — 장편과 단편이 섞인 정밀 샘플

이번엔 빙과를 해 볼 건데, 빙과는 장편 에피와 단편 에피가 섞여 있고, 캐릭터들의 말투가 일정해서 아주 정확한 샘플이라 생각했다.

빙과는 미스터리/일상물이다. 22화 중 장편 아크 3개(氷菓 사건 1~5, 영화 사건 8~11, 칸야제 12~17)와 단편 미스터리 7개(6, 7, 18~22)가 섞여 있다. 같은 작품 안에서 **"매화 닫는 단편"과 "이월하는 장편"의 직접 비교**가 가능하다.

**사전 계량: 캐릭터 코사인 유사도 (JP)**

일본어 원문 자막에 화자 태그가 달려 있어서(（奉太郎）, （える） 등), 캐릭터별 대사를 정밀하게 분리할 수 있었다.

| | avg cosine | 해석 |
|---|---|---|
| 메인 4명 평균 | **0.934** | 서로 매우 비슷하게 말함 |
| 奉太郎↔里志 | **0.984** (최고) | 사실상 동일 문체 |
| 전체 20명 평균 | 0.896 | 단역만 독자적 |

그리고 대사 수(log 글자수)와 Shannon entropy의 상관계수 **r = 0.888**. 말투가 안 변하니까 엔트로피가 순전히 샘플 사이즈의 함수로만 수렴한다. 캐릭터 구분이 "어떻게 말하느냐"가 아니라 "뭘 말하느냐"인 작품.

| 200~500자 | 6.091 |

zlib(EN): avg **0.402** (z=-0.12) — 각본 베이스라인과 거의 동일. direction 비율 34.2%.

**3중 Payoff 분석 (JP 원문, GPT-5.2)**

아크 구조를 하드코딩하고, GPT-5.2에게 아크 컨텍스트를 알려준 뒤 분석했다. 3중 체크: 매화 닫힘 + 장편 아크 종결 + 시리즈 종결.

| 아크 | 화수 | 유형 | 아크내 avg | 피날레 | 아크 종결 |
|------|------|------|-----------|--------|-----------|
| 氷菓 사건 | 1~5 | 장편 | 71.4 | **92** | definitive |
| 대충 지내도 되지 않나 | 6 | 단편 | — | **92** | definitive |
| 정체를 알 수 없는 소포 | 7 | 단편 | — | **92** | definitive |
| 영화 사건 | 8~11 | 장편 | 61.2 | **85** | partial |
| 칸야제 | 12~17 | 장편 | 62.5 | **88** | partial |
| 連峰は晴れているか | 18 | 단편 | — | **92** | definitive |
| 心あたりのある者は | 19 | 단편 | — | **88** | definitive |
| 遠まわりする雛 | 20 | 단편 | — | **92** | definitive |
| バレンタイン | 21 | 단편 | — | **92** | definitive |
| 遠い雷鳴 | 22 | 단편 | — | **92** | definitive |

사이클 완결: **22/22 (100%).**

| | avg | min | σ |
|---|---|---|---|
| **전체 22화** | **73.5** | 35 | 17.4 |
| **단편 (7화)** | **91.4** | 88 | — |
| **장편 에피 (15화)** | 65.1 | 35 | — |
| **장편 피날레 (3화)** | **88.3** | 85 | — |

단편 7화가 전부 88~92. 미스터리 제시→추리→해결이 1화 안에 완벽하게 닫힌다. 빙과의 단편 에피소드는 지금까지 측정한 모든 작품의 어떤 구간보다 높다.

장편 중간 화는 52~72로 내려간다. 셋업이 필요하니까. EP8(35)이 유일한 50 미만 — 영화 아크 오프너로 순수 셋업 화다. **하지만 장편 피날레에서 확실하게 닫는다.** EP5(92), EP11(85), EP17(88).

이월 주기를 보면:

> EP8(**35**) → EP9(55) → EP10(70) → EP11(**85**)

코드기아스 R2E10(40)→R2E11(70)→R2E13(88)과 같은 패턴이다. 빚을 졌으면 이자 붙여서 갚는다. 빌린 만큼 돌려준다. 그리고 마지막 EP22(92)에서도 미스터리를 닫는다.

시리즈 종결 점수: **78.** 오레키의 성장과 치탄다와의 관계는 완전히 닫지 않고 여운으로 남긴다. "言いかけて止める/言いかけて飲み込む"(말하려다 멈추고/말하려다 삼키고). 이건 원작이 미완결이라 그런 것이기도 하지만, 빙과라는 작품의 톤이기도 하다.

그리고 이건 **"못 닫은 것"이 아니라 "안 닫은 것"**이다. 빙과는 단편에서 7번 연속 88~92로 닫는 걸 보여줬다. 닫는 능력이 있다는 걸 증명한 작품이 마지막에 안 닫은 거다. EP22의 에피소드 payoff는 92. 미스터리는 닫았다. 근데 오레키가 치탄다에게 "経営的戦略眼を俺が..."(경영적 전략안을 내가...)라고 말하려다 삼킨다.

닫을 수 있다는 걸 보여준 뒤에 안 닫는 건 **선택**이고, 닫을 수 있는지 증명 못 한 채 안 닫는 건 **실패**다. 그리고 오레키가 말을 삼키는 것 자체가 이 캐릭터의 본질이다 — "やらなくてもいいことなら、やらない"(안 해도 되는 건 안 한다). 에너지 절약주의자가 고백을 삼키는 게 이 캐릭터의 프랙탈 최소 단위 그 자체. 안 닫은 것이 곧 닫은 것이다. 오레키는 아직 오레키라는 선언.

**프리렌과의 비교: 같은 H_norm, 다른 달성 방법**

빙과와 프리렌은 H_norm이 거의 같다 (0.719 vs 0.712). 하지만 내용물이 다르다.

| | 프리렌 | 빙과 |
|---|---|---|
| min | **52** | 35 |
| max | 92 | 92 |
| 진폭 | **40** | **57** |
| 장편 아크 오프너 | EP18 = **52** (11화 아크) | EP8 = **35** (4화 아크) |
| 바닥 안 뚫는 비결 | 셋업 화에서도 닫음 | 단편이 평균을 끌어올림 |

프리렌은 1급 마법사 시험이 11화짜리 장편인데, 그 오프너(EP18)가 52다. 시험 등록 + 팀 배정 + 전략 수립이라는 "셋업 of 셋업" 화인데도 한 번도 바닥을 안 뚫는다. 빙과는 4화짜리 아크 오프너에서 35까지 내려간다. 빙과가 H_norm을 낮게 유지하는 건 단편 91.4가 장편 EP8(35)을 상쇄하기 때문이고, 프리렌은 진짜로 전 화가 52 이상이기 때문이다.

이게 프리렌의 DFA α=0.957(거의 완벽한 1/f 프랙탈)이 나오는 이유다. **셋업 구간이 존재하지 않는다.** 11화짜리 장편에서도.

**6작품 비교**

| 작품 | avg | min | H_norm | 전략 |
|---|---|---|---|---|
| **빙과** | **73.5** | 35 | **0.719** | 단편은 매화 닫기, 장편은 피날레에서 닫기 |
| Frieren | 72.3 | 52 | 0.712 | 매화 닫기 |
| FMA:B | 69.0 | 45 | 0.779 | 이월 후 즉시 폭파 |
| Code Geass | 66.1 | 40 | 0.802 | 이월 후 즉시 폭파 |
| Steins;Gate | 64.8 | 35 | 0.864 | 장기 축적 후 일괄 폭파 |
| Platinum End | 61.9 | 35 | 0.940 | 패턴 없음 |

빙과의 H_norm = 0.719. 프리렌(0.712)과 사실상 동급. "잔잔하지만 매번 닫는다"가 정확히 같은 패턴.

빙과가 증명한 것: **같은 작품 안에서 단편(91.4)과 장편(65.1)이 공존할 때, 장편 피날레(88.3)가 단편 수준으로 복귀한다.** 셋업은 장편에서 필요할 수 있다. 하지만 아크가 끝날 때 확실하게 닫는다. 그리고 마지막에도 닫는다.

결국 **타이밍 차이**지, 무조건 "닫아라"다.

| 작품 | 최대 이월 | 닫는가? |
|---|---|---|
| 프리렌 | 0화 | ● |
| 빙과 단편 | 0화 | ● |
| 빙과 장편 | 3~5화 뒤 피날레 | ● |
| 하가렌/코드기아스 | 1~2화 뒤 폭발 | ● |
| 슈타게 | 12화 빚 → 후반 폭발 | ● |
| 플래티넘 엔드 | 패턴 없음 | ✗ |

#### 각본에서도 같은 원리: 9개 이벤트는 전부 "닫아라"였다

Part 3의 각본 분석에서, 영화를 씬 단위로 나누고 9개 이벤트 타입(SETUP, INCITING, GOAL, ACTION, OBSTACLE, DISASTER, DISCOVERY, RESOLUTION, EMOTION)으로 분류한 뒤, 11개 메트릭을 계산했다([JOURNEY_prediction.md](JOURNEY_prediction.md) 시도 11~17).

명작과 망작을 분리한 메트릭들:

| 메트릭 | Cohen's d | 방향 |
|---|---|---|
| repeat_ratio | **+0.789** | 명작이 높음: 같은 유형이 연속 반복 |
| arc_shift | **-0.662** | 명작이 낮음: 전후반 구조 일관 |
| setup_front | **-0.648** | 명작이 낮음: 셋업을 앞에 안 몰아넣음 |
| bi_entropy | **+0.523** | 명작이 높음: 전환 패턴이 다양 |

그리고 시퀀스를 직접 보면:

```
Inception (8.8):           GOA ACT OBS ACT ACT RES DIS ACT ACT ACT...
From Justin to Kelly (2.0): SET GOA INC GOA EMO SET EMO OBS EMO EMO EMO...
The Room (3.6):            SET SET SET SET INC INC EMO EMO EMO SET EMO...
```

인셉션은 매 씬마다 GOAL→ACTION→OBSTACLE→RESOLUTION을 닫는다. 망작은 SETUP과 EMOTION만 반복하면서 안 닫는다.

**이게 씬 레벨의 payoff score다.** 그때는 이걸 몰랐다. 9개 이벤트 분류가 필요하고, 11개 메트릭을 계산해야 하고, Cohen's d를 비교해야 한다고 생각했다. 근데 결국 11개 메트릭이 전부 같은 얘기를 하고 있었다:

- repeat_ratio ↑ = 씬 안에서 밀어붙인다 = **닫는다**
- setup_front ↓ = 셋업을 앞에 안 몰아넣는다 = **매 씬에서 닫는다**
- emotion_ratio ↓ = 감정만 반복 안 한다 = **닫는다**
- action_ratio ↑ = 행동한다 = **닫는다**

씬마다 payoff_score 하나만 물어보면 됐다. 에피소드별로 한 것처럼.

| 스케일 | 좋은 작품 | 나쁜 작품 |
|---|---|---|
| **씬** (영화) | GOA→ACT→OBS→RES 닫힘 | SET SET EMO EMO 안 닫힘 |
| **에피소드** (TV) | payoff 52+ 매화 닫힘 | payoff 35 붕 뜸 |
| **아크** (TV) | 피날레 85+ 폭발 | 방향 없이 들쭉날쭉 |
| **시리즈** | 마지막에 닫힘 (혹은 의도적으로 안 닫힘) | 못 닫힘 |

영화든 드라마든 프랙탈이다. 스케일만 다를 뿐이다.

## 결론: 좋은 전개의 원리

진리는 단순했다.

**닫아라.**

- 씬 안에서 닫아라 (영화)
- 에피소드 안에서 닫아라 (TV)
- 아크 끝에서 닫아라 (장편)
- 시리즈 끝에서 닫아라 (전체)

이월은 허용된다. 장편에서 셋업은 필요하다. 하지만 빚을 졌으면 갚아라. 타이밍만 다를 뿐, 원칙은 하나다.

| 작품 | 평균 payoff | H_norm | 전략 |
|---|---|---|---|
| Frieren | 72.3 | 0.712 | 매화 닫기. 셋업 구간이 존재하지 않음 |
| 빙과 | 73.5 | 0.719 | 단편은 매화, 장편은 피날레에서. 안 닫은 건 선택 |
| FMA:B | 69.0 | 0.779 | 이월 후 즉시 폭파. 다음 화에 이자까지 갚음 |
| Code Geass | 66.1 | 0.802 | 이월량 많지만 반드시 갚음 |
| Steins;Gate | 64.8 | 0.864 | 12화 빚 → 후반 프리렌급으로 전부 회수 |
| Platinum End | 61.9 | 0.940 | 패턴 없음. 빚을 갚다가 또 지고 갚다가 또 짐 |

각본 2,433편에서 시작해서, 영화 씬 레벨의 9-이벤트 분류를 거쳐, 애니메이션 6작품 에피소드별 payoff 분석까지 — 전부 같은 결론이었다.

**setup+payoff now → plus carry-forward.**

지금 닫고, 그러면서 이월도 한다. 닫는 게 먼저고, 이월은 부산물이다. 이게 프랙탈 서사의 원리이고, 좋은 전개의 전부다.

애니메이션은 자막만 있으면 끝이다. 자막 타임스탬프 하나가 direction:dialogue ratio를, 대사 텍스트가 zlib 정보밀도를, 그리고 전문을 LLM에 보내면 에피소드별 payoff score까지 나온다. 그리고 payoff 하나로 좋은 전개와 나쁜 전개가 갈린다.
