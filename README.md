# chori-scratchpad

> Hirameki를 정리해두는 곳.

새벽에 갑자기 떠오른 아이디어를 코드로 옮겨보는 공간입니다.
장난처럼 시작했지만, 검증이 필요한 것들이 많고, 언젠가 다른 프로젝트나 솔루션의 씨앗이 될 수도 있는 것들을 모아둡니다.
